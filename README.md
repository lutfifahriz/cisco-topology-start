# Star Topology Simulation

Simulasi jaringan menggunakan topologi Star pada Cisco Packet Tracer.

## Deskripsi
Project ini menggunakan:
- 1 Server
- 1 Switch
- 5 PC

Server dikonfigurasi sebagai DHCP Server dan seluruh PC menggunakan DHCP Client.

## Konfigurasi
- DHCP Server
- Automatic IP Addressing
- Centralized Switch Connection

## Implementasi
Topologi Star merupakan topologi yang paling umum digunakan pada jaringan modern seperti kantor dan sekolah.

## Kelebihan
- Mudah dikembangkan
- Mudah melakukan troubleshooting
- Jika satu kabel rusak, perangkat lain tetap berjalan

## Kekurangan
- Ketergantungan pada switch utama
- Jika switch rusak, jaringan terganggu
